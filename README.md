
Colibri Dashboard Frontend

Dashboard de React para gestión de transporte compartido

Instalación:
npm install

Ejecutar:
npm start (http://localhost:3000)

Componentes:
- Header: Navegación principal
- Dashboard: Panel de inicio
- Map: Visualización de viajes en mapa
- Transactions: Tabla de transacciones
- FinanceReports: Recargas y retiros
- Analytics: Gráficas y análisis
- Users: Gestión de usuarios

Librerías principales:
- React Router: Navegación
- Axios: Llamadas API
- Highcharts: Gráficas
- Leaflet + React Leaflet: Mapas
- CSS Modules: Estilos

Variables de entorno (.env):
- REACT_APP_API_URL: URL base de API

Build:
npm run build
